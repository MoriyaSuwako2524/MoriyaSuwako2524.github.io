0.5. Check List

Check List

This checklist is designed for new members joining the CCATS Group. It helps you verify that all required servers and software are properly configured before starting any computational work.

1. Supercomputers

In the CCATS group, we primarily use two clusters: PETE and OSCER, hosted respectively by Oklahoma State University (OSU) and the University of Oklahoma (OU). These two clusters are largely interchangeable. For example, if PETE is under maintenance, you can usually submit your jobs to OSCER instead.

However, due to differences in system architecture and software environments, the recommended usage is as follows:

  • PETE: for CPU-based tasks
    (e.g., Q-Chem quantum chemistry calculations, classical molecular dynamics simulations)
  • OSCER: for GPU-based tasks
    (e.g., machine learning, QM/MM simulations)

Account Request

You can use the following link to apply:

PETE: https://hpcc.okstate.edu/account-request.html
OSCER: https://www.ou.edu/oscer/support/accounts/new_account

Before applying, inform Dr. Shao that you plan to request an account.
Typically, your account will be approved within one week.
Do not apply from outside the United States.

If you encounter any issues during registration or setup, please ask other group members for assistance.

2. IQmol

Our group primarily performs quantum chemical calculations using Q-Chem. You do not need to install Q-Chem locally, since we maintain multiple customized versions on the supercomputers for method development.
If you need access to a specific version, please ask one of the group members.

If you are a graduate student who plans to compile or modify Q-Chem yourself, refer to the section
Q-Chem Basic
for detailed instructions on downloading, compiling, and configuring Q-Chem.

2.1 Installing IQmol

To visualize Q-Chem results, you should install IQmol on your personal computer (Windows or macOS recommended).
If you plan to run molecular dynamics simulations, you may also need VMD for trajectory visualization.

Unless you are very familiar with Linux, it is better to use Windows or macOS for your local setup.

You can download IQmol from its official website:
https://www.iqmol.org/

Download the latest version, extract the archive, and install it following the on-screen instructions.

2.2 Verification Checklist

After installation, please verify that:

  1. IQmol launches normally.
  2. You can build a molecule using the graphical interface.
  3. You can save the molecule file correctly.
  4. You can open and visualize Q-Chem output files.

For more detailed guidance, see the dedicated section:
IQmol

2.3 Known Issues

macOS security warning:
On macOS, IQmol may show a warning that the developer’s certificate cannot be verified.
To fix this, go to System Preferences → Security & Privacy, and allow IQmol to run.

macOS File saving issue:
Some macOS users have reported that IQmol cannot save files properly.
Installing an older version of IQmol may resolve this issue.

If you encounter other installation or runtime problems, searching through Google or ChatGPT(or other AI) can help you find effective solutions.

3. Server Access and File Transfer (Optional)

Since PETE and OSCER are remote computing clusters, you need to access them through SSH or similar remote connection methods. In principle, you can use CMD or PowerShell on Windows, or the command line on macOS or Linux, to connect to the servers and use the scp command for file transfer.

However, for beginners, using the command line for all operations can be inconvenient and may take some time to become familiar with. Therefore, it is recommended to use dedicated software tools to simplify these tasks.

For Windows users:
XShell: remote server access and command-line operations.
Xftp or WinSCP: file transfer between your local computer and the server.

For macOS users:
FileZilla: file transfer between your local computer and the server.

These tools provide a graphical interface that makes connecting to remote servers and managing files much easier.
